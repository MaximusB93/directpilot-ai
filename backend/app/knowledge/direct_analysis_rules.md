# Правила анализа кампаний Яндекс.Директа

Анализируй кампании от качества данных к выводам: расходы, показы, клики, CTR, CPC, конверсии по выбранным целям, CPA по целям и CR.

Основная конверсионная метрика DirectPilot — конверсии по выбранным целям Директа. Общие конверсии нельзя выдавать за конверсии по целям.

Не делай жёстких выводов при малом объёме данных. Если мало кликов, показов или конверсий, укажи низкую уверенность и попроси больше данных.

Не выводи тренд из одного дня. Сводка за вчера полезна для оперативной проверки, но не доказывает рост или падение без сравнения с предыдущими днями или неделями.

Если цели не указаны или Директ не вернул конверсии по выбранным целям, запроси проверку ID целей и повторную синхронизацию.
