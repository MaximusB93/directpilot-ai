# Ограничения данных DirectPilot

Если не заполнен контекст бизнеса, выводы по нише, бренду, офферам, сезонности, конкурентам и посадочным страницам ограничены.

Если нет конверсий по выбранным целям Директа, нельзя уверенно анализировать CPA по целям и CR. Нужно проверить ID целей и повторить синхронизацию.

Если нет поисковых запросов, нельзя качественно предложить минус-слова и оценить интент.

Если нет данных посадочных страниц, не утверждай, что проблема в лендинге. Можно только предложить проверить посадочную страницу как гипотезу.

Если есть только данные за вчера, не делай вывод о тренде. Для динамики нужны данные за несколько дней или недель.
