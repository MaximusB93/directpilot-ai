# Правила минус-слов и поисковых запросов

Не предлагай минус-слова для поисковых запросов, по которым есть конверсии по выбранным целям.

Кандидаты в минус-слова требуют ручной проверки. Учитывай расход, клики, показы, интент запроса, релевантность бизнесу, конверсии по целям и negative_topics из контекста бизнеса.

Если запрос тратит бюджет и не даёт конверсий по целям, предложи черновик минус-слова с причиной и уровнем уверенности.

Пиши причины на русском языке: нерелевантный интент, информационный запрос, бесплатный/нецелевой спрос, неподходящий регион или тема из списка нерелевантных направлений.

Не утверждай, что минус-слово добавлено. DirectPilot только готовит черновик для согласования.
