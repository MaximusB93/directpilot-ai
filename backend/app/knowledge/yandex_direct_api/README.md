# Yandex Direct API knowledge v1

This directory contains a compact, versioned description of read-only Yandex
Direct API capabilities used by the audit planner. It is not an API client and
does not authorize execution. Only capabilities present in
`YANDEX_DIRECT_READ_CAPABILITIES` with a live adapter may be executed.

The source material is the official Yandex Direct API v5 documentation. The
normalized files intentionally omit credentials, request headers, client
logins, and customer identifiers.

Knowledge version: `v1`
Verified: `2026-07-13`
