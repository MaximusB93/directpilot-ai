# Глоссарий метрик DirectPilot

Расход — сумма затрат на рекламу за выбранный период.

Показы — сколько раз объявления были показаны пользователям.

Клики — сколько раз пользователи перешли по объявлениям.

CTR — доля кликов от показов. Помогает оценить привлекательность объявления и релевантность спроса.

CPC — средняя стоимость клика.

Конверсии по целям — достижения выбранных целей Директа. Это основная конверсионная метрика DirectPilot.

CPA по целям — расход, делённый на конверсии по выбранным целям.

CR — конверсия из клика в выбранную цель.

ДРР/ROMI можно оценивать только если есть выручка, маржа или ценность лида. Без revenue-данных не выдумывай окупаемость.
